# 🔍 Plagiarism Checker

A web-based plagiarism detection system built with Django that uses Cosine Similarity algorithm to compare texts and documents.

## Features

- ✅ Check text for plagiarism against web content
- ✅ Upload and check .txt, .docx, and .pdf files
- ✅ Compare two texts or documents for similarity
- ✅ Cosine similarity algorithm for accurate results
- ✅ Responsive web interface

## Tech Stack

- Django 3.1.2
- NLTK for text preprocessing
- Scikit-learn for TF-IDF and Cosine Similarity
- PyPDF2 for PDF processing
- python-docx for Word documents
